# 02 · Machine Learning & NLP

Hands-on ML engineering work spanning classical models, neural architectures, and transformers. These projects build the modeling muscle I'll carry into healthcare ML — clinical NLP, patient-sentiment analysis, and predictive modeling on health data.

## Projects

- **Emotion_Classification_Partner_Project/** — Classifying emotional states from text (directly transferable to mental-health and patient-sentiment applications).
- **NER Tagging Model/** — Named entity recognition using HMM + Viterbi decoding, with a full data-exploration and validation pipeline.
- **NER_Tagging_FFNN_RNN/** — Neural re-implementation of NER with feed-forward and recurrent architectures, custom tokenizer, and training loop.
- **Seagull Transformers - Humor Generation Model/** — Transformer-based text generation experiment.
- **BookBeats** — Song-recommendation system built from book descriptions. See `BookBeats_Project.txt` · [Repo](https://github.com/ay283/bookbeats) · [Live site](http://4300showcase.infosci.cornell.edu:4180)

## Skills exercised

Feature engineering, tokenization, sequence modeling (HMM, RNN, Transformers), training loops in PyTorch, evaluation pipelines, and full-stack ML product deployment (BookBeats).
