# 05 · Mobile & Product Experiments

Smaller product-focused builds where I've explored mobile, UX, and ML-powered recommendation from a maker's perspective. They're a reminder that great ML only matters when it ships inside a product people actually want to use.

## Projects

**daredevil — iOS Motivation App**
Repo: https://github.com/aileen0101/daredevil

A mobile app designed to nudge users toward action by turning motivation into a lightweight daily experience.

**BookBeats — Song Recommendation System**
Repo: https://github.com/ay283/bookbeats
Live site: http://4300showcase.infosci.cornell.edu:4180

A full-stack ML product that recommends songs based on the mood and themes of a book. Built end-to-end: data pipeline, model, backend, and deployed web UI.
