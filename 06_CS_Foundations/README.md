# 06 · CS Foundations

The coursework and practice projects that built the core CS fluency I rely on in every project above — data structures, OOP, functional programming, and systems-level Python.

## Subfolders

### `Java/` — Data Structures & OOP
- **CMSµ** — Course management system data model.
- **GUI Mouse-Clicker Game** — Event-driven GUI programming.
- **McDiver Sewer Navigation** — Graph algorithms and pathfinding.
- **PhD Genealogy Tree** — Tree data structures and traversal.
- **Spreadsheet Merging Program** — File I/O, parsing, and data merging.

### `OCaml/` — Functional Programming
- **Chat Text Generator Application** — Probabilistic text generation.
- **OCaml DNA Tree Predictor** — Recursive tree-based bioinformatics prototype.
- **Search Document Engine Application** — Inverted index + query engine.

### `Python/`
- **Color Models GUI** — Interactive color-space exploration.
- **Currency Exchange Application** — API-driven calculator with robust input handling.
- **Imager Application** — Image-processing pipeline (filters, transformations).
- **Planetoids GUI Game Application** — Asteroids-style game with physics and collision.
- **Turtle Graphics Application** — Recursive and iterative graphics generation.

## Why These Are Here (Not Hidden)

These projects taught me how to *think* — recursion, invariants, graph traversal, clean interfaces, functional composition. Every healthcare ML model I build sits on top of this foundation.
